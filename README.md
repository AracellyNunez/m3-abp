# Analizador de Constantes Hematimétricas y Anemias

## Introducción

Este proyecto es una aplicación web desarrollada con HTML, CSS y JavaScript que permite ingresar datos clínicos básicos de un paciente y obtener un análisis preliminar de sus constantes hematológicas. Su finalidad es mostrar cómo se pueden combinar conceptos de programación web con la lógica de un problema clínico de forma sencilla, clara y funcional.

Este trabajo fue desarrollado como parte de una entrega para un curso, con el propósito de demostrar el uso de formularios, manipulación del DOM, validaciones de datos y lógica de programación aplicada a un contexto real.

## Objetivo del proyecto

El objetivo principal de esta aplicación es facilitar la evaluación de valores hematológicos básicos, como:

- Hemoglobina (Hb)
- Hematocrito (Hto)
- Glóbulos rojos (RBC)

A partir de estos datos, la aplicación calcula indicadores como:

- VCM (Volumen Corpuscular Medio)
- HCM (Hemoglobina Corpuscular Media)
- CHCM (Concentración de Hemoglobina Corpuscular Media)

Además, proporciona una clasificación morfológica sugerida y muestra alertas cuando los valores ingresados pueden ser poco coherentes o indicar un posible problema clínico.

## Funcionalidades principales

La aplicación permite:

✅ Ingresar datos del paciente mediante un formulario.
✅ Calcular automáticamente los índices hematológicos.
✅ Clasificar de forma general el estado hematológico del paciente.
✅ Mostrar alertas en caso de valores anormales o inconsistentes.
✅ Registrar los pacientes analizados en una tabla histórica.

## Tecnologías utilizadas

- HTML5: para la estructura de la interfaz.
- CSS3: para el diseño y estilo visual.
- JavaScript: para la lógica del programa, cálculos y manejo de eventos.

## Estructura del proyecto

El proyecto está organizado de la siguiente manera:

- index.html: archivo principal que contiene la estructura de la página.
- assets/css/styles.css: archivo de estilos para la apariencia de la interfaz.
- assets/js/index.js: archivo con la lógica de cálculo, validaciones y funciones del programa.

## ¿Cómo usar la aplicación?

1. Abrir el archivo index.html en un navegador web.
2. Completar los campos del formulario con los datos del paciente.
3. Presionar el botón para calcular los resultados.
4. Revisar la información mostrada en pantalla.
5. Observar el historial de pacientes registrados en la tabla inferior.

## Explicación breve del funcionamiento

Cuando el usuario ingresa los datos, el sistema realiza una serie de validaciones para comprobar que los valores sean coherentes. Luego, aplica las fórmulas correspondientes para determinar los índices hematológicos y presenta una clasificación sugerida. Finalmente, guarda la información del paciente en un registro histórico para su consulta.
